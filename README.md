# z1miner

### Information
z1miner is a Monero (XMR) CPU miner, api based by coinhive.com

### Requirement
* node.js v8.4.0 or higher

### Installation
```console
user@terminal:~$ npm update
user@terminal:~$ chmod +x z1miner

```

### Usage
```console
user@terminal:~$ z1miner <site_key> <secret_key> <thread>
```
